== Description ==

There is a great news for people in the business of jewelry making, jewelry designing, selling, polishing etc. We present Alankar Jewelry Store WordPress Theme that can help jewelry designers, jewelry showroom owners, traders, wholesalers to showcase their business without spending too much time or money. We’re sure you will love the idea of taking your jewelry business online displayed on this attractive and functional theme. 
Alankar Jewelry Store WordPress Theme is an extraordinary looking, functional, easy to manage, responsive and secure theme that has been designed within guidelines. This design is an outcome of long hours of research, in-depth understanding of industry, its customers, and their triggers. That is the reason when you have a close look at the layout and overall vibe you will be convinced that it will enable you to fit every important aspect of your business on it. 
The colors have been chosen to suit the niche. However if you wish to change background colors to match with your brand colors Alankar Jewelry Store WordPress Theme allows you to do it. Dashboard is where you will find all the instructions to change images, colors, text and more. This theme also lets you create a blog section to post blog through which you will engage with your customers and build following. 
You may be a start-up or an established jewelry designer; this theme will work for you. You can manage it yourself without coding skills or tech knowledge. You can create catalogues, upload logo, social media links, and everything else that you have wished your online presence to display about your business. You won’t find a more attractive or a more affordable deal elsewhere. You won’t need to keep an in-house digital team or an individual to manage your operations as you will be able to do it all by yourself via your mobile phone.


For Homepage Setup:
====================

1. Go to Dashboard >> Pages >> Add New, Create a new page and name it as "Home" then select the template "Custom Home Page" and publish it.

2. Go to Dashboard >> Settings >> Reading, Select the option of Static Page, now select the page you created to be the homepage.

For Header Section:
====================

1. Go to Dashboard >> Appearance >> Customizer >> Theme Settings >> Header Section, add topbar text, add phone text, add phone number and publish it.

For Slider:
============

1. Go to Dashboard >> Pages >> Add New, Create a new Page and name it, add the content, serviced image and publish the page. Repeat this step for other sliders.

2. Go to Dashboard >> Appearance >> Customizer >> Theme Settings >> Slider Settings, Click the checkbox of Show/Hide Slider, select the pages of the slider that you have created and publish it.

For Service Section:
====================

1. Go to Dashboard >> Posts >> Category, add new category for service, name it and publish it.

2. Go to Dashboard >> Posts >> Add New, Create a new Post and name it, assign the service category and publish the post. Repeat the step for other service posts.

2. Go to Customizer >> Theme Settings >> Service Section, select the service category that you have created, you can also change the number of post to show & icons of service and publish it.

For Service Section:
====================

1. Go to Dashboard >> Posts >> Category, add new category for services, name it and publish it.

2. Go to Dashboard >> Posts >> Add New, Create a new Post and name it, assign the service category and publish the post. Repeat the step for other service posts.

2. Go to Customizer >> Theme Settings >> Service Section, add section title, select the service category that you have created and publish it.

For Footer Section:
====================

1. Go to Dashboard >> Appearance >> Widgets, here you can add widgets to the footer widgets areas, which can be seen in footer.

For Copyright Text:
====================

1. Go to Customizer >> Theme Settings >> Footer Text, add footer text here and it also have an option to show/hide back to top.
